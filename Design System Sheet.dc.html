COMPONENTS + STATES
Every interactive element is one "chunky" recipe: gradient fill, 3px dark outline of the same hue, 6px bottom edge (toy depth), 2px inner top highlight. Pressed = sink 3px, bottom edge collapses to 3px.

FRAME + CARD ANATOMY
Window frame: dark outer ring (4px) → lime rim (3px) → deep-green body. Radius 20. Purple rim for Rebirth.
Header: 78px banner, category gradient at 105°, dark stud tiling at 7%, 4px dark bottom edge.
Section container: #0E2A15, 2.5px near-black outline, radius 16, padding 18.
Card: hue gradient (light→base), 3px dark-of-hue outline, radius 14, 4px bottom drop of the same dark.
Well (art/input): darkest tone, inner shadow, radius 10 — content sits *into* the panel.
SPACING + RADIUS LOGIC
SPACING — 8PX GRID (halves allowed)
Panel edge padding: 20px — nothing ever touches the frame
Section padding: 18px · card padding: 12–14px
Gutters: cards 12–14px · sections 16px · rows 10px
Row height: 56–58px · tab height: 52–56px
Icon↔text gap: 9–10px · label↔control: min 16px
RADIUS — SQUARER OUTSIDE, ROUNDER INSIDE
Window frame: 20px · section container: 16px
Cards: 14px · buttons/tabs: 12px
Pills/dropdowns/toggles: 10px · wells: 10px
Badges/chips/keycaps: 8–9px
Never pill-shaped (999px) buttons. Radius never exceeds the parent's radius.
QUALITY BAR — NON-NEGOTIABLES
✓ Outlined text uses layered round shadows — never sharp miter joins
✓ One chunky recipe for every interactive element, all four states defined
✓ Saturated fills only on cards, tabs and CTAs — structure stays deep green/purple
✓ Badges never overlap text, never rotate, never clip outside cards
✓ Every screen: 20px edge padding, 8px-grid gaps, no drifting alignment
✓ Stud texture at ≤7% opacity, headers and large surfaces only
✓ Rebirth swaps hue (purple), never shape — same anatomy everywhere
✓ Text minimum 12px; row labels 15.5px; nothing cramped against a border
✓ Hidden tabs and future menus reuse these exact recipes — no new one-off styles
✓ Pressed states physically sink (3px) — every click feels mechanical
